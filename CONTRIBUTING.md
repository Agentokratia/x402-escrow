# Contributing

Thanks for your interest in contributing to x402-escrow!

## Getting Started

```bash
git clone https://github.com/agentokratia/x402-escrow.git
cd x402-escrow
npm install
cp .env.example .env.local
npm run dev
```

## Pull Requests

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Run tests (`npm test`)
5. Submit a PR

## Code Style

- Run `npm run format` before committing
- Run `npm run lint` to check for issues

## Questions?

Open an issue or discussion on GitHub.
